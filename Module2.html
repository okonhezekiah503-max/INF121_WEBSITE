Module 2: HyperText Markup Language (HTML)

The building block of every web page.

Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create and organize the content of web pages. HTML uses elements called tags to define headings, paragraphs, images, links, tables and forms.

Main Topics

  • HTML Document Structure
  • HTML Tags and Attributes
  • Headings and Paragraphs
  • Lists
  • Tables
  • Forms
  • Images and Multimedia

Sample HTML Structure

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>

<body>

</body>
</html>

Example of an HTML Table

Tag Purpose
<h1> Main Heading
<p> Paragraph
<img> Display Image
<a> Create Hyperlink

Importance of HTML

  1. Creates the structure of web pages.
  2. Works together with CSS and JavaScript.
  3. Supports multimedia integration.
  4. Allows creation of forms for user input.
  5. Provides semantic meaning to content.
