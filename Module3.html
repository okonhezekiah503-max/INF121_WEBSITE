Module 3: Cascading Style Sheets (CSS)

Styling and designing modern web pages.

Introduction

Cascading Style Sheets (CSS) is a stylesheet language used to control the appearance and layout of HTML documents. It helps developers create attractive, responsive, and professional websites.

Main Topics

  • Introduction to CSS
  • Ways of Applying CSS
  • CSS Selectors
  • Text Formatting
  • Fonts and Colors
  • Borders, Margins and Padding
  • Using CSS with HTML

Methods of Applying CSS

Method Description
Inline CSS Applied directly inside an HTML element.
Internal CSS Written inside the <style> tag.
External CSS Stored in a separate .css file and linked to HTML.

Advantages of CSS

  1. Improves website appearance.
  2. Separates design from content.
  3. Makes websites easier to maintain.
  4. Creates responsive layouts.
  5. Allows code reuse across multiple pages.

Simple CSS Example

body{
    background:#eef2f7;
    font-family:Arial;
}

h1{
    color:#0b3d91;
}

p{
    font-size:18px;
}
